## 📌 Описание программы

**Random Task Generator** - это desktop-приложение для генерации случайных задач, которое помогает разнообразить ваш день и организовать выполнение различных дел. Программа имеет "живую" статистику, которая обновляется в реальном времени при каждом действии.

### 🎯 Основные возможности:

| Функция | Описание |
|---------|----------|
| 🎲 **Генерация задач** | Случайный выбор задачи из списка одним нажатием |
| 📋 **Управление задачами** | Добавление и удаление задач с выбором типа |
| 📊 **Живая статистика** | Обновляется автоматически при каждом действии |
| 📜 **История** | Сохранение всех сгенерированных задач |
| 🔍 **Фильтрация** | Просмотр истории по типам задач |
| 💾 **Работа с JSON** | Сохранение и загрузка истории в JSON формат |
| 📖 **Инструкция** | Встроенное руководство пользователя |

### 🏷️ Типы задач:
- 📚 **Учёба** - учебные задачи (чтение, изучение, практика)
- 🏃 **Спорт** - спортивные активности (зарядка, пробежка, тренировки)
- 💼 **Работа** - рабочие задачи (отчёты, совещания, проекты)

### Установка:

```bash
# Скачайте файл task_generator.py
# Запустите в терминале:
python task_generator.py
Никаких дополнительных библиотек не требуется!

📖 Как пользоваться
1️⃣ Генерация случайной задачи
Просто нажмите кнопку "СГЕНЕРИРОВАТЬ ЗАДАЧУ" - программа случайным образом выберет задачу из списка. Задача отобразится на экране и автоматически добавится в историю.

2️⃣ Добавление новой задачи
Введите название задачи в поле "Новая задача"

Выберите тип задачи (учёба, спорт, работа)

Нажмите кнопку "ДОБАВИТЬ"

Задача появится в списке, статистика обновится

3️⃣ Удаление задачи
Выберите задачу в списке (кликните на неё)

Нажмите кнопку "УДАЛИТЬ ВЫБРАННУЮ ЗАДАЧУ"

Задача исчезнет, статистика обновится

4️⃣ Фильтрация истории
Выберите тип задачи в выпадающем списке

Нажмите "ПРИМЕНИТЬ" - история покажет только задачи выбранного типа

Нажмите "СБРОСИТЬ" - фильтр отключится

5️⃣ Работа с JSON
СОХРАНИТЬ В JSON - сохранить историю в файл tasks_history.json

ЗАГРУЗИТЬ ИЗ JSON - загрузить историю из выбранного файла

ОЧИСТИТЬ ИСТОРИЮ - удалить все записи из истории

6️⃣ Живая статистика
Статистика обновляется автоматически при каждом действии и показывает:

Общее количество задач в списке

Распределение задач по типам

Количество генераций по типам

Процентное соотношение

Самую популярную категорию

Количество активных дней

Среднее количество задач в день

text

text

4. Следите за статистикой: сколько задач решено
Пример 3: Спортивный режим
text

3. Генерируйте случайное упражнение
4. Сохраните прогресс через JSON
